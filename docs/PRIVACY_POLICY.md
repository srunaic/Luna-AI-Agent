## 개인정보처리방침 (Privacy Policy)

본 문서는 Nanodoroshi / Anticode(이하 “서비스”)의 개인정보 처리에 대한 기본 방침입니다.  
실제 배포 시 **운영자 정보/연락처/도메인**을 채워서 웹에 게시해야 합니다.

### 1) 운영자(Controller)
- **운영자/개발자**: (여기에 기입)
- **연락처 이메일**: (예: `mailto:admin@example.com`)
- **서비스 URL**: (여기에 기입)

### 2) 수집하는 정보
서비스는 기능 제공을 위해 아래 정보를 처리할 수 있습니다.
- **계정 식별 정보**: username, uid(서비스 내부 식별자)
- **프로필 정보(선택)**: 닉네임, 아바타 URL
- **채팅/커뮤니티 데이터**: 채팅 메시지, 채널 정보, 초대/강퇴/차단 기록
- **기기/접속 정보(제한적으로)**: user agent 등(디버그/호환성 확인 목적)
- **푸시 알림 정보(옵트인)**: Web Push 구독 정보(endpoint, p256dh, auth)

### 3) 수집/이용 목적
- 계정 식별 및 기능 제공(로그인/세션)
- 채널/친구/초대/강퇴/차단 등 커뮤니티 기능 제공
- 보안/악용 방지/장애 대응(제한적 로그/메타데이터)
- 사용자가 동의한 경우 푸시 알림 발송

### 4) 보관 및 삭제
- 서비스 운영에 필요한 기간 동안 보관하며, 이용자가 삭제를 요청하거나 법령상 의무가 없는 경우 지체 없이 삭제합니다.
- 푸시 알림은 사용자가 “푸시 끄기” 또는 기기/브라우저 설정에서 해제 시 구독 정보가 비활성화/삭제될 수 있습니다.

### 5) 제3자 제공/처리위탁
서비스는 인프라 제공을 위해 아래 외부 서비스를 이용할 수 있습니다.
- **Supabase**: 데이터베이스(PostgreSQL), 스토리지, 엣지 함수
- **Cloudflare Pages**: 정적 웹 호스팅/배포

### 6) 이용자 권리
이용자는 본인 정보에 대해 열람/정정/삭제/처리정지 등을 요청할 수 있습니다.  
요청 방법: 위 “연락처 이메일”로 문의.

### 7) 보안조치
- 비밀키/서비스키 등은 클라이언트에 포함하지 않고 서버 측(Secrets)에서 관리합니다.
- 데이터 손실 대비를 위한 백업을 운영합니다: `docs/BACKUP.md`

### 8) 변경
본 방침은 서비스 운영/법령 변경에 따라 개정될 수 있으며, 변경 시 서비스 내 또는 게시 페이지를 통해 고지합니다.
