# Luna AI Agent

로컬 에이전트 개발환경 (Electron + Django + ChromaDB)

- 최신 릴리즈: v0.1.39

## 핵심 기능
- 메모리/RAG: ChromaDB 저장 + 검색
- 관리자 UI: http://127.0.0.1:8000/api/memory/admin/ui/
- 로컬 Django 자동 구동: 127.0.0.1:8000
